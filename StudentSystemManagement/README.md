# STUDENT SYSTEM MANAGEMENT

## Giới thiệu
Chương trình quản lý sinh viên được xây dựng bằng C++ sử dụng WinAPI và GDI+.
Hệ thống hỗ trợ đăng nhập, đăng ký tài khoản, phân quyền người dùng và quản lý bởi Admin.

## Công nghệ sử dụng
- Ngôn ngữ: C++
- Giao diện: WinAPI, GDI+
- Lưu trữ dữ liệu: File text (users_data.txt)
- Môi trường: Windows
- IDE: Visual Studio Code

## Chức năng chính
- Đăng nhập tài khoản
- Đăng ký tài khoản mới
- Phân quyền người dùng (Admin, Student, Lecturer)
- Admin duyệt và xóa tài khoản
- Giao diện đồ họa bằng GDI+
